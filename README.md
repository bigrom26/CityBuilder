# CityBuilder

## Instructions to run project on Mac :

### Make sure to install all necessary dependencies:
     brew install qt
     brew install glew
     brew instlall glfw
     
### Use Qmake command to create MakeFile :
     qmake Playground.pro

### Use make command to create executable :
    make 

### Open application :
    open playground.app/
